Ventana de Inicio de Sesión

Función Principal

Autentica usuarios y los redirige según su rol:

  • Administrador: Crear una parada nueva
  • Responsable: gestión de las paradas
  • Peregrino: editar datos y exportar el carnet

Instrucciones de Uso

1. Ingresar Credenciales

Campo superior: Escribe tu nombre de usuario o email

Campo inferior: Introduce tu contraseña (se muestra oculta)

Presiona Enter en el campo de contraseña para iniciar sesión rápidamente

2. Botones Disponibles

Iniciar Sesión: Valida tus datos y accede al sistema

Registrar Peregrino: Abre el formulario para nuevos usuarios

3. Comportamiento del Sistema

  • Credenciales correctas → Redirección automática al menú correspondiente
  • Credenciales incorrectas → Muestra mensaje de error
