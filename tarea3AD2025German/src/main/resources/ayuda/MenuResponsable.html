Cómo usar el Menú Responsable

Este menú principal te permite acceder a las funciones principales de la aplicación para gestionar alojamientos y estancias.

Opciones del menú:

  • Gestion
    • Sellar/Alojar: Abre una ventana para sellar o alojar a un peregrino.
      Atajo: Ctrl + A
    • Exportar Estancias: Permite exportar la información de las estancias registradas.
      Atajo: Ctrl + E
  • Help
    • About: Información sobre la aplicación (no implementado en esta versión).
  • Salir
    • Cerrar Sesión: Cierra la sesión actual y regresa a la pantalla de login.
      Atajo: Ctrl + S

Uso de atajos de teclado:

Para agilizar tu trabajo, puedes usar los siguientes atajos de teclado en cualquier momento mientras estás en la aplicación:

  • Ctrl + A para abrir la ventana de Sellar/Alojar.
  • Ctrl + E para abrir la ventana de Exportar Estancias.
  • Ctrl + S para cerrar sesión y volver al login.

Selecciona la opción deseada del menú o usa el atajo para acceder rápidamente a la función correspondiente.
