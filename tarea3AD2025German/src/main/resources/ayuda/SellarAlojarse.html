Cómo usar la ventana Sellar/Alojarse

Esta ventana te permite registrar la estancia de un peregrino en tu parada. Puedes sellar y alojar a un peregrino y seleccionar el tipo de estancia que corresponde.

Pasos para sellar o alojar a un peregrino:

  1. Selecciona el nombre del peregrino:
    En el menú desplegable "Nombre del Peregrino", elige el peregrino que deseas registrar.
  2. Elige el tipo de estancia (opcional):
    Puedes seleccionar Estancia Normal o Estancia VIP.
    Nota: Solo puedes seleccionar una opción a la vez. Si no seleccionas ninguna, se registrará la visita sin tipo específico.
  3. Confirma el registro:
    Haz clic en el botón Confirmar para guardar la estancia.
  4. Salir de la ventana:
    Si deseas cancelar o salir, presiona el botón Salir.

Qué sucede al confirmar:

  • Se guarda la estancia con la fecha y hora actuales.
  • Si la estancia es VIP, se incrementa el contador de estancias VIP en el carnet del peregrino.
  • Se registra la relación entre el peregrino y la parada actual.
  • Se muestra una alerta confirmando el tipo de estancia guardada o un error si seleccionaste ambas opciones.

Errores comunes:

  • Seleccionar ambas opciones de estancia (Normal y VIP) simultáneamente no está permitido.
  • No seleccionar ningún peregrino antes de confirmar puede impedir el registro.

Utiliza esta ventana para gestionar correctamente la estancia de los peregrinos y llevar un control actualizado en la aplicación.
