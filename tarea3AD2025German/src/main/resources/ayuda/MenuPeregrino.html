Guía de Usuario

Ventana: Menú de Peregrino

Esta ventana permite al usuario peregrino gestionar su carnet, modificar su información personal, y cerrar sesión.

Menú: Gestión

  • Exportar carnet (Ctrl+E)
    Genera un informe PDF del carnet del peregrino y lo guarda también como fichero XML en la carpeta ficheros/.
    • El informe PDF se muestra con JasperViewer.
    • El archivo XML incluye datos del carnet, peregrino, paradas y estancias.
  • Modificar Peregrino (Ctrl+M)
    Abre una ventana para modificar los datos del usuario peregrino actual (nombre, nacionalidad, etc.).

Menú: Salir

  • Cerrar Sesión (Ctrl+S)
    Finaliza la sesión actual del usuario y redirige a la pantalla de inicio de sesión.
